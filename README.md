# car_detect

## 1차 학습 결과
사용 데이터 : [Kaggle](https://www.kaggle.com/sshikamaru/car-object-detection)

밑에 이미지는 학습 후 detect를 시도하였을 때 모습을 보여준다.
해당 모델에서는 object는 차량만을 가정하고 detect하였다.
